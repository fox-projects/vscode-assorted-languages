# vscode-assorted-languages

Never worry about syntax highlighting for miscellaneous files again!

## Usage

Syntax highlighting for

- asl
- ASM x86_64
- autotools (autoconf / automake)
- AWK
- Bats
- CISCO
- Desktop Files
- dircolors
- dsdl
- EDK2 misc (`.fdf`, `.dsc`, `.dec`, `.inf`, `.uni`, `.vfr`)
- EJS
- env
- Expect
- FlatBuffers
- gnuplot
- HAML
- Handlebars
- HLA
- i3
- infokey
- inputrc
- IO
- jq
- JSON5
- jsonl
- Linker Scripts
- Log
- Logstash
- M3U
- Linker Maps
- Ninja
- Odin
- gettext / po
- Protobuf
- QlikView
- Stylus
- sxhkdrc
- systemd
- T4
- terminfo
- Thrift
- Typo3
- VCard
- Xresources / Xdefaults
- Yarn Lockfiles
- Zone Files (Bind)
- Vagrantfile
