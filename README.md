# vscode-assorted-languages

Never worry about syntax highlighting for miscellaneous files again!

## Usage

Snippets for

- EJS
- Handlebars
- infokey

Syntax highlighting for

- AWK
- Desktop Files
- dircolors
- EJS
- env
- Handlebars
- HLA
- i3
- infokey
- inputrc
- JSON5
- Ninja
- gettext
- [STOL](https://ntrs.nasa.gov/citations/19780014834) (Systems Test and Operation Language)
- sxhkdrc
- systemd
- terminfo
- Xresources
