# vscode-assorted-languages

Never worry about syntax highlighting for miscellaneous files again!

## Usage

Snippets for

- EJS
- Handlebars

Syntax highlighting for

- AWK
- Desktop Files
- dircolors
- EJS
- env
- Handlebars
- i3
- JSON5
- Ninja
- gettext
- sxhkdrc
- systemd
- terminfo
- Xresources
